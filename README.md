# Notreal™ Engine
A graphics engine that has totally ~~un~~notreal capabilities. This is a wip hobby graphics engine written with C++, OpenGL, and GLFW. I am following the wonderful [LearnOpenGL](https://learnopengl.com/) tutorial!
